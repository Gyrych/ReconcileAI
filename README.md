# n8n VS Code Extension Workspace

本仓库包含 VS Code 插件项目：将 n8n 的 Web 界面集成到 VS Code。

- 插件代码与文档见：`vscode-n8n/`
  - 使用与配置说明：`vscode-n8n/README.md`
  - 可行性研究报告：`vscode-n8n/docs/FEASIBILITY.md`

快速开始：

1. 进入子目录并安装依赖：
   ```bash
   cd vscode-n8n
   npm install
   npm run build
   ```
2. 使用 VS Code 打开本仓库，按 F5 启动调试。
